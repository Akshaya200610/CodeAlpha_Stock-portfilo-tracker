📈 Stock Portfolio Tracker

A simple Python-based application that helps users calculate the total value of their stock investments. The program uses predefined stock prices, accepts user input for stock names and quantities, computes individual investments, and stores the total portfolio value in a text file.

🚀 Features

- Track investments in multiple stocks.
- Uses a dictionary to store stock prices.
- Calculates individual and total investment values.
- Handles invalid stock names.
- Saves the portfolio summary to a "portfolio.txt" file.
- Beginner-friendly and easy to understand.

🛠️ Technologies Used

- Python 3

📂 Project Structure

▶️ How to Run

1. Clone the repository:

git clone https://github.com/your-username/Stock-Portfolio-Tracker.git

2. Navigate to the project folder:

cd Stock-Portfolio-Tracker

3. Run the program:

python stock_portfolio_tracker.py
